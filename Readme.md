# EduMarketing React App

## Project Description
EduMarketing is a simple React application focused on online education and digital marketing.  
The project was created as a learning exercise to practice React, routing, reusable components, responsive design, and clean code principles.

The design is inspired by Moosend and EduDemos.

---

## Main Page Description
The home page displays a list of courses related to digital marketing, UX/UI design, and React development.  
The courses are rendered dynamically from a JSON array using reusable components and props.

The layout is fully responsive using Flexbox and media queries.

---

## Project Structure
The project is organized following a modular React structure:

- components: Reusable UI components (Header, Footer, CourseCard, Map)
- pages: Main application pages (Home, About, Contact)
- data: JSON-like data files
- public/images: Optimized images for web use

---

## Contact Page

The contact page includes:

- Contact information (email and location)
- An interactive map using Leaflet
- A simple contact form with name, email, and message fields
